Overview

This project was built as a potential website for a friend’s band – “Plastic Terras”. Due to its purpose being more focused on entertainment than business-related, the website needed a more playful look compared to the clean style you commonly find in professional websites. I achieved this by using a striking static background throughout. This brought of ensuring that text was visible on the busy backdrop while also allowing the background the be shown.
The website utilises flexbox and the Bootstrap grid system in tandem to achieve a responsive layout and maintain a intelligible appearance no matter the size of the viewport. The homepage makes use of a plug-in called scrollify.js which allowed to create a large landing area which snaps to the next section on the page as soon as the user scrolls.
The website includes separate pages for the bands music, upcoming shows and mock merchandise. The navbar also contains external links their media accounts such as Spotify, Soundcloud, Facebook and Instagram. The homepage includes an iframe to import one of the band’s music videos from YouTube.
